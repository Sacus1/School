Objectif : apprendre à contrôler des mouvements rigides avec les outils d'algèbre matricielle.

Le compte rendu devra comporter la mise en forme du problème.

En cas de besoin, une documentation des classes est fournie.

En fin de TP vous devrez remettre un compte-rendu et une archive zip ou tgz (aucun autre format n'est accepté) comportant les fichiers Java que vous aurez réalisés (Matrix.java, Exo3.java et Exo4.java). Les codes fournis devront pouvoir être compilés sans erreur.

Exercice 3 : Résolution matricielle de mouvements

Partie 3a : Opérations matricielles

Compiler et exécuter Exo3.java. Le but est de refaire la trajectoire de la cabine sur le câble par composition de matrices homogènes.

Dans une première étape, compléter l'implantation des méthodes suivantes de la classe Matrix (convention vecteur colonne du cours) :
- void set (float tx, float ty, float tz) : transforme la matrice en matrice de translation.
- void set (char axis, float cosAngle, float sinAngle) : transforme la matrice en une matrice de rotation; les valeurs possibles pour le paramètre axe sont 'X', 'x', 'Y', 'y', 'Z' ou 'z'; le cosinus et le sinus de l'angle sont précalculés.
- void set (char axis, float angleDegre) : transforme la matrice en matrice de rotation. Attention, l'angle est exprimé en degrés.
- void set (float scale) : transforme la matrice en matrice d'homothétie uniforme selon les 3 axes.
- void mult (Matrix mat) : transforme la matrice par sa multiplication par mat.
- void multLeft (Matrix mat) : transforme la matrice par la multiplication de mat par la valeur initiale de la matrice.
Attention, il est recommandé de tester systématiquement chacune des méthodes.

Partie 3b : Mise en oeuvre sur un mouvement plus complexe

Par rapport à l'exercice du TP précédent, le trajet de la cabine doit maintenant s'accompagner d'un mouvement pendulaire autour du câble sous la forme de deux cycles sinusoïdaux d'amplitude 40 degrés. La mise en forme du problème comporte une rotation supplémentaire dont l'angle est une fonction sinus de l'avance.

Exercice 4 : Incréments de rotation

Compiler et exécuter Exo4.java. Dans cet exercice, la cabine doit effectuer un mouvement circulaire continu pour suivre le câble.

Laisser tourner quelques tours, puis noter ce que vous observez dans le compte-rendu.

Analyser le fichier Exo4.java pour trouver l'explication du phénomène et la reporter dans le compte-rendu. La géométrie de la scène est augmentée de méthodes d'instance supplémentaires :
- float[] cableCenter () : position 2D du centre du cercle d'appui du câble;
- float cableRadius () : rayon du câble.

Proposer et implanter une solution pour résoudre le problème sans remettre en cause le principe de l'incrément matriciel.
